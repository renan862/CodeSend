## Olá, seja bem-vindo!

Este é um projeto pessoal desenvolvido por hobby. Portanto, podem existir bugs ou funcionalidades em desenvolvimento.

## Sobre o projeto

A proposta é criar um backend capaz de:

- Gerenciar sessões do WhatsApp  
- Realizar disparos em massa a partir de listas (ex: Excel)  
- Utilizar variáveis dinâmicas nas mensagens  
- Aplicar intervalos aleatórios entre envios (para maior naturalidade)  

## Stack

# Backend:

- Node.js
- Express.js
- PostgreSQL

# Front:

- React

O projeto ainda está em desenvolvimento e novas funcionalidades serão adicionadas em breve.
