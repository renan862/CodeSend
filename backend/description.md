## Descrição do backend

No backend, será implementada a API responsável por atender o frontend e disponibilizar rotas para o front

Neste projeto, será utilizado o WppConnect para o envio inicial de mensagens em massa no WhatsApp. Os nomes dos destinatários serão recebidos por meio de uma planilha de dados nos formatos `.xls` ou `.xlsx`.

## Stack

- Node.js
- Express.js
- PostgreSQL
